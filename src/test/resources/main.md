# Заголовок

Обычный текст с **жирным** и *курсивом*.
                
Можно комбинировать: ***жирный курсив***, **жирный с *курсивом* внутри**.
                
Ссылка: [текст ссылки](https://example.com)
                
Код: `inline code` в строке.
                
```java
public class Example {
    public static void main(String[] args) {
        System.out.println("Hello!");
    }
}
```
                
## Список
                
* Первый элемент
* Второй элемент с **жирным**
* Третий элемент
                
> Цитата с важной информацией
                
Специальные символы: . ! - нужно экранировать.
