*Заголовок*

Обычный текст с *жирным* и _курсивом_\.

Можно комбинировать: _*жирный курсив*_, *жирный с _курсивом_ внутри*\.

Ссылка: [текст ссылки](https://example.com)

Код: `inline code` в строке\.

```java
public class Example {
    public static void main(String[] args) {
        System.out.println("Hello!");
    }
}
```

*Список*

- Первый элемент
- Второй элемент с *жирным*
- Третий элемент

>Цитата с важной информацией_

Специальные символы: \. \! \- нужно экранировать\.
